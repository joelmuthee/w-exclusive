# W Exclusive Wellness & Fitness - Website Pitch

A premium, high-performance landing page designed for W Exclusive at Westgate Shopping Mall.

## Features
- **Ultra-Modern Design:** Gold & Navy aesthetic tailored for luxury wellness.
- **Bi-directional Scroll Reveals:** Smooth fade-in-up animations that respond to scroll direction.
- **Premium Modal System:** Custom booking inquiry modal with body-scroll locking.
- **Optimized for Mobile:** Custom intersection thresholds and flexible typography for small screens.
- **Social Media Ready:** Built-in Essence Protocol for perfect WhatsApp/Instagram sharing previews.

## Tech Stack
- HTML5 / Vanilla CSS
- Tailwind CSS (via CDN)
- Swiper.js (Reviews Slider)
- FontAwesome 6 (Icons)
- Intersection Observer API (Animations)
